Financial Named Entity Recognition: How Far Can LLM Go?

Yi-Te Lu, YinTong Huo

The 2025 FinNLP-FNP-LLMFinLegal Joint Workshop, in conjunction with COLING 2025

Abstract:

The surge of large language models (LLMs) has revolutionized the extraction and analysis of crucial information from a growing volume of financial statements, announcements, and business news. Recognition for named entities to construct structured data poses a significant challenge in analyzing financial documents and is a foundational task for intelligent financial analytics. However, how effective are these generic LLMs and their performance under various prompts are yet need a better understanding. To fill in the blank, we present a systematic evaluation of state-of-the-art LLMs and prompting methods in the financial Named Entity Recognition (NER) problem. Specifically, our experimental results highlight their strengths and limitations, identify five representative failure types, and provide insights into their potential and challenges for domain-specific tasks.

A Mobile Edge Computing Based QoS Adaptation Architecture for Smart Healthcare

基於行動邊緣運算架構之智慧醫療服務品質調適

Yi-Te Lu, Hao-Ying Cheng, Yung-Li Hu, I Chang Tsai, Yennun Huang, Sy-Yen Kuo

The 2023 Taiwan Academic Network Conference (TANET)

Abstract:

Mobile edge computing is a vital network service architecture in the modern information industry. It excels in integrating IoT device computing power and ensuring the reliability of interactions between edge computing servers and IoT devices, enabling better quality of service. Computation offloading is a widely used technique within MEC, which addresses limitations such as limited edge device computational resources and high energy consumption. By shifting high-computational and energy-consuming tasks to nearby edge servers, computation offloading reduces computation latency and extends device lifespans. Furthermore, with the rise of smart healthcare and the urgency caused by the pandemic, smart healthcare with the internet of medical things has gained prominence. This research project proposes to enhance the quality of service of smart healthcare within the Mobile Edge Computing framework through computation offloading. Using Taiwan Taoyuan International Airport as an example, the study explores how our approach can improve the accuracy, real-time performance, and reliability of smart emergency medical services.

QoS-Aware Adaptation of Edge-Enabled Metaverse Service in Telemedicine Applications

邊緣元宇宙服務之服務品質調適於遠距醫療應用

Yi-Te Lu, Hao-Ying Cheng, Yung-Li Hu, I Chang Tsai, Yennun Huang, Sy-Yen Kuo

Poster presented at 2023 19th Taiwan Conference on Software Engineering (TCSE)

Abstract:

The metaverse is an emerging technology in the information industry, requiring the handling of massive data flow and computation. This places specific demands on aspects such as computational performance and latency, which impact the quality of service (QoS). Mobile edge computing addresses these QoS requirements of the metaverse by orchestrating computational resources. Therefore, services built on mobile edge computing must possess QoS-awareness and the ability to adapt during operation to maintain their QoS. This research develops a metaverse service based on a mobile edge computing framework, incorporating QoS-awareness and adaptive capabilities. Remote healthcare services are used as the experimental scenario to evaluate and verify the QoS-awareness and adaptation efficiency of the proposed edge computing framework.

A Design of Resilient Software Framework on Edge Computing

具復原力之邊緣運算服務軟體架構設計

Hao-Ying Cheng, Yi-Te Lu, Yung-Li Hu, Chuan-Ming Liu, Yennun Huang

The 2022 Taiwan Academic Network Conference (TANET)

Abstract:

Edge computing services are an important service model in the field of Internet of Things (IoT) applications. By deploying service software on edge computing nodes near application environments, these services improve response immediacy and reduce transmission latency. However, in IoT scenarios, the increase in the number of sensing devices or malicious attacks can cause fluctuations in data traffic, affecting the processing efficiency of the services. Resilient edge computing services possess adaptive capabilities, allowing them to allocate system resources to accommodate changes in the current application environment.
The decentralized and resource-constrained nature of edge computing environments presents challenges to achieving resilience. This research proposes a service architecture design methodology for resilient edge computing services based on a decentralized self-adaptive system and a microservices architecture. Using a scenario where fluctuations in data traffic are caused by changes in the number of devices in an IoT environment, we establish an experimental edge computing environment for implementation and verification. The results demonstrate the operational effectiveness of resilient edge computing services in adaptive system resources flexibly to adapt to changes in the application environment within a decentralized and resource-constrained system.

Design and Implementation on Context-Aware Service Software in Pervasive Edge Computing Environment

普及邊緣運算環境下情境感知服務軟體之設計與實作

Zong-Lun Li, Yi-Te Lu, Yung-Li Hu, Yennun Huang, Sy-Yen Kuo

Poster presented at 2021 17th Taiwan Conference on Software Engineering (TCSE)

Honorable Mention Poster Award

Abstract:

Pervasive edge computing is an edge computing implementation with a focus on integrating the computational capabilities of IoT devices, responding to dynamic changes in application scenarios, and ensuring reliable interactions between edge servers and IoT devices. This approach allows artificial intelligence technologies to be deployed more adaptively and flexibly according to task scenarios, offering greater scalability. Due to the decentralized nature of Pervasive edge computing environments, the high heterogeneity among edge nodes, the extensive use of encapsulation and third-party software services, the need for mobility-aware data processing, and the highly dynamic nature of task environments, software for pervasive edge computing applications must possess context-awareness. Such software should adapt to changes in application scenarios and system states during runtime. This research explores the design and implementation of context-aware service software architecture in pervasive edge computing environments. It proposes methods for software architecture modeling and programming to achieve context-awareness in these environments. The study validates its findings through the simulation of a small-scale autonomous vehicle application.

Decentralized Complex Event Processing Engine applied on Fog Computing

應用於霧運算之非集中性複雜事件處理引擎

Research Center for Information Technology Innovation, Academia Sinica

Advisor: Prof. Yung-Li Hu, Prof. Yen-Nun Huang

Abstract:

Complex Event Processing (CEP) is an event-based data processing technology based on logical inference, and has applied on Internet of Things (IoT) applications widely. Due to traditional CEP for IoT applications was executed on cloud systems, so that amount of network resource consumption and high response latency occurred on sensing data transmission. Fog computing, a novel technology move computation and storage resources to network edge of IoT devices, can reduce network resource consumption on data transmission and response latency from remote sites. CEP executed on fog computing systems can save network resource consumption and improve performance on response latency, and enhance interaction between IoT devices. Therefore, in this research project, a decentralized CEP engine is be proposed which can realize event-driven IoT coordination based on microservice technology. The result of this research can realized event-driven IoT services efficiently, and adapt system execution state depended on application and system context.
